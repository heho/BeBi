Simple mostly stylefree components that enable you to structurize your website.

the Grid

A versatile grid system that should cover all needs. It utilizes a semantic approach, if you need a six column wide row you create a div with the classes 'six', 'wide' and 'row' than create your nested grid with the same syntax a two columns wide column has the classes 'two', 'wide' and 'column'

some more basics

You would not like to miss these basic structure elements. Containers can contain stuff and are centered, they can be fluid aswell and are responsive by default. Also containers come in sizes: 'mini', 'tiny' and 'small'. Modules just ensure the spacing around content. Just add the corresponding classes to a div. Experiment a bit to see which can be mixed in which cant.

.container
.fluid.container
.mini.container
.tiny.container
.small.container
.module

Typografic Elements are basic page parts that structurise the pages content.

headers
